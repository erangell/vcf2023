SOUND CHECK

Computers store sounds and music as files. When your Web Browser plays a sound, it downloads a file from a location on the internet.

The Web Browser uses a Plug-In program to play the sound. Apple QuickTime and Windows Media Player are examples of Plug-Ins. The Plug-In may look as follows:

Windows
Macintosh
When the Web Browser finds a web page with a sound file, it looks for a Plug-In program to play the sound.

If the Plug-In is found, the control panel for the Plug-In will be displayed (as shown above). You can then click the Play button (the right pointing triangle) to play the sound. Note: Some sounds may play right away, depending on how the web page was designed.

If the Web Browser cannot find the Plug-In, it will either display a broken link, or a link to a webpage for downloading the Plug-In.

Note: Your Web Browser may display a Security Warning when it finds a web page with a Plug-In. You will need to follow the instructions to Allow the program to run.
