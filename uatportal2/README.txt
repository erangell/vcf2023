This ZIP file contains an archive of the Music Basics Online system developed for the University of Delaware in 2005 and used until 2014, when it was decommmissioned.  It ran under Sun Solaris with PHP, MySQL, HTML, Javascript and Java.

This archive only contains the webpages and Java applets.  The PHP code is included, but non-functional.

To launch the web page: open mbo/mbo.html in a web browser.
